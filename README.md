# ☕ Estudos de Java

Repositório destinado ao armazenamento de códigos, exercícios e projetos desenvolvidos durante meus estudos de lógica de programação e desenvolvimento com Java.

## 🛡️ Sobre Mim
Atualmente sou estudante de **Defesa Cibernética** na Faculdade Estácio de Sá. Este repositório faz parte do meu processo de aprendizado em linguagens de programação, buscando fortalecer a base técnica necessária para a área de segurança.

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Java (JDK 25)
* **IDE:** Spring Tools Suite (STS) / Eclipse
* **Ambiente:** WSL2 (Ubuntu) no Windows
* **Versionamento:** Git & GitHub

## 📁 Estrutura do Projeto
Os arquivos estão organizados na pasta `src` e incluem:
* `Main.java`: Ponto de entrada para testes rápidos.
* `ex34.java`: Exercícios específicos de lógica.
* `products.java`: Práticas de Orientação a Objetos.
* `Program.java`: Implementações de lógica de negócio.
